This extension allows execution of a custom Python script for a Makro Action in the Session Handling rules of Burp.

To use, type or paste a Python script into the "Makro Script" tab, and configure the Makro to your needs. The script will be executed when you configure it as a Makro action in the Project Options - Sessions - Session Handling Rules (Add one) - Rule Action. Simply consider the "Makro Script" tab to be the code of the performAction function as described on the API description of Burp for the ISessionHandlingAction.

The following variables are defined in the context of the script:

  • extender
  • callbacks
  • helpers
  • currentRequest
  • macroItems
  • All interfaces (IBurpCollaboratorClientContext, etc.) from the current (May, 5 2017) Burp API

This plugin is only a very simple modification of Marcin Wielgoszewski's Python Scripter, simply swapping the processHttp API with the performAction API.
